## Research Protocol
1. Scope: Define question precisely. Set token budget.
2. Gather: Check codebase first, then docs, then web (max 5 searches).
3. Analyze: Compare on complexity, performance, maintainability.
4. Report: Table format. Option | Pros | Cons | Recommendation. Under 200 words.
