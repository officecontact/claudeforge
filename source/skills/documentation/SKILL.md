## Doc Standards
**README:** Name + one-liner > Quick Start (<30s) > Features > Usage > API > Config > Contributing.
**API docs:** Endpoint > Method > Params (table) > Response > Example (runnable).
**Comments:** WHY not WHAT. No commented-out code. TODO format: `// TODO(name): desc`.
