# Prompt Cheatsheet — Save Tokens

## Be Specific
❌ "make this better"
✅ "refactor auth.js — extract token refresh, add error handling"

## Save Tokens
1. "Output only code. No explanations."
2. "Follow the pattern in src/components/UserCard.tsx"
3. "Update error handling in auth.js, user.js, and api.js" (batch)
4. "Use the researcher agent to find how payments work" (delegate)
5. "Only modify src/auth/" (scope)

## Anti-Patterns
- "Can you help me with..." → Just state the task
- Asking Claude to explain before doing → Just do it
